java装箱与拆箱

发表于 2019-09-05 分类于 java基础

包装类：基本数据类型使用方便，但没有对应方法来操作这些数据，因而使用一个类，在类中定义操作数据的方法，这些类叫做包装类

简单来说，java装箱就是基本数据类型转换成包装类类型，拆箱就是包装类类型转换成基本数据类型

对应关系如下

基本数据类型 包装类
byte Byte
boolean Boolean
short Short
int Integer
float Float
double Double
char Character
